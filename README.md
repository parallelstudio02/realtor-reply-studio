# Realtor Reply Studio

A simple web app for turning client messages and rough realtor replies into natural replies with practical tone options.

## Features

- Client message box and rough reply box
- WhatsApp and email formats
- Email subject generation
- Realtor tones including friendly, professional, analytical, education, negotiation, follow up, reassuring, concise, objection handling, seller update and passive aggressive
- Output avoids contractions, hyphenated phrasing and comma before `and`
- Mobile compatible
